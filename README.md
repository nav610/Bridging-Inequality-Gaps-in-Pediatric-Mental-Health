# Bridging Inequality Gaps in Pediatric Mental Health Through Primary Care

This repository contains the publicly available dataset accompanying the study
*"Bridging Inequality Gaps in Pediatric Mental Health Through Primary Care."*

The data has been de-identified and contains **no Protected Health Information (PHI)**.
It is shared to support reproducibility, secondary analysis, and further research into
equitable pediatric mental health care.
